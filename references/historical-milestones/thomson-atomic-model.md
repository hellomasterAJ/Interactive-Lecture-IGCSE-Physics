# ⚛️ Thomson's Atomic Model — The Discovery of the Electron

## 📜 Overview

In **1897**, the British physicist **Joseph John (J.J.) Thomson** (1856–1940) at the Cavendish Laboratory, Cambridge, discovered the **electron** — the first subatomic particle ever identified. This discovery shattered the 2,000-year-old belief that the atom was indivisible (the very word "atom" comes from Greek *atomos* = "uncuttable").

Thomson then proposed the **"plum pudding" model** (or "watermelon model") of the atom: a uniformly positive sphere with negatively charged electrons embedded inside like plums in a pudding.

> *"I can see no escape from the conclusion that [cathode rays] are charges of negative electricity carried by particles of matter."*
> — **J.J. Thomson**, 1897

| Key fact | Detail |
|----------|--------|
| **Who** | Joseph John Thomson (1856–1940), Cavendish Professor of Experimental Physics, Cambridge |
| **When** | 30 April 1897 (announced at the Royal Institution) |
| **What** | Discovered the **electron** using cathode-ray tubes |
| **Key measurement** | **Charge-to-mass ratio (e/m)** of the electron = 1.76 × 10¹¹ C/kg |
| **Why it matters** | Proved atoms are **divisible** — the first elementary particle; won the **Nobel Prize in Physics** (1906) |
| **Model** | "Plum pudding" model (1904): positive sphere + embedded electrons |
| **Overturned by** | **Ernest Rutherford** (1911) — gold foil experiment → nuclear model |

## 🔬 The Experiment — Core Content

### The Cathode-Ray Tube (CRT)

```
[High-voltage source]                    [Electrodes: anode and cathode]
        |                                        │
        ▼                                        ▼
    ┌─────────────────────────────────────────────────┐
    │                                                 │
    │      ┌──── (+)                     (−) ────┐    │
    │      │                                       │    │
    │      ▼                                       │    │
    │   ╔═══╗       ╔══════════╗      ╔═══════╗   │    │
    │   ║Anode║      ║ Deflectn ║      ║Fluore-║   │    │
    │   ║ (+)║      ║ Plates   ║      ║scent  ║   │    │
    │   ╚═══╝       ╚══════════╝      ║Screen ║   │    │
    │                   │             ╚═══════╝   │    │
    │    ←cathode rays→│→→→→→→→→→→→      ↑         │    │
    │                   │          green spot       │    │
    └─────────────────────────────────────────────────┘
          Evacuated glass tube (high vacuum)
```

**Apparatus components:**
1. **Cathode** (−) — emits cathode rays when heated by high voltage
2. **Anode** (+) — accelerates the rays
3. **Collimating slits** — produce a narrow beam
4. **Electric deflection plates** — apply E-field to bend the beam
5. **Magnetic coils** — apply B-field to bend the beam
6. **Fluorescent screen** — where the beam produces a visible green spot

### What Thomson Did

| Step | Observation | What It Proved |
|------|-------------|----------------|
| 1 | Applied high voltage → glass tube glowed green at the screen end | Cathode rays travel in straight lines |
| 2 | Placed an object in the path → **shadow** appeared on the screen | Rays travel in straight lines (like light) |
| 3 | Applied an **electric field** across the tube → beam **deflected** toward the + plate | Cathode rays are **negatively charged** |
| 4 | Applied a **magnetic field** → beam deflected (proportional to B) | Rays respond to magnetic fields |
| 5 | **Balanced** E-field and B-field so beam was undeflected | v = E/B (velocity of the particles) |
| 6 | Measured the beam deflection under E alone | **e/m = v/(B²·r)** |

---

### 🧮 The Key Measurement — Charge-to-Mass Ratio (e/m)

Thomson measured the electron's charge-to-mass ratio by **balancing electric and magnetic forces**:

**Step 1: Find the velocity (v)**
When electric force = magnetic force:
```
eE = evB   →   v = E/B
```
By adjusting E and B until the beam is undeflected, v is known.

**Step 2: Deflect with magnetic field only**
Turn off E → the beam curves in a circular path:
```
evB = mv²/r   →   e/m = v/(Br)
```
Substituting v = E/B:
```
e/m = E/(B²r)
```

**Thomson's result:**
```
e/m = 1.76 × 10¹¹ C/kg
```

| What this meant | Explanation |
|-----------------|-------------|
| The particle is **over 1,800× lighter** than a hydrogen ion (proton) | Electrons were far, far lighter than the lightest atom |
| The **same** e/m was measured for **every cathode material** (Cu, Fe, Al, Pt) | These particles were **universal** — they existed in **all atoms** |
| Either the charge was **huge** or the mass was **tiny** | (Millikan later proved it was the mass that was tiny — 9.11 × 10⁻³¹ kg) |

> 💡 **Key inference:** Atoms of all elements contain identical electrons → atoms are **not indivisible** — they have structure!

## 🔍 Detail — The Plum Pudding Model (1904)

### What Thomson Proposed

> "The atom consists of a sphere of uniformly distributed **positive electricity**, in which **negative electrons** are embedded — much like **plums in a pudding** (or **seeds in a watermelon**)."

```
                      ┌─────────────────────────┐
                     │      Positive "dough"     │
                     │    (uniform charge +)     │
                     │                           │
                     │    ●  ●       ●           │
                     │         ●                 │
                     │  ●            ●           │
                     │       ●                    │
                     │  ●         ●      ●       │
                     └─────────────────────────┘
                     Electrons (−) embedded
```

### The Model's Predictions

| Feature | Details |
|---------|---------|
| **Positive charge** | Uniformly distributed throughout the atom (the "pudding") |
| **Electrons** | Point-like negative charges inside the positive sphere |
| **Net charge** | Neutral atom: total e⁻ charge = total positive charge |
| **Size of atom** | ≈ 10⁻¹⁰ m (1 Å) |
| **Number of electrons** | Thomson estimated ≈ number of atomic mass (e.g. hydrogen ≈ 1, oxygen ≈ 16?) |
| **Why electrons stay** | Electrostatic attraction to the positive sphere |
| **What holds the atom** | Electrical forces (no new force needed) |
| **Light emission** | Electrons oscillate about equilibrium positions → emit EM waves |

### What It Got Right

- ✅ Atoms are **divisible** (contain electrons)
- ✅ Atoms contain **negative particles** (electrons) and **positive charge**
- ✅ Atoms are **electrically neutral** overall
- ✅ The electron is **universal** — found in all elements

### What It Got Wrong (Rutherford 1911)

| Plum pudding predicted | Rutherford found |
|----------------------|-----------------|
| Positive charge spread throughout the atom | Positive charge concentrated in a **tiny nucleus** (1/10,000 the atom's size) |
| Alpha particles should pass through with **slight deflection** | Most passed through, but **1 in 8,000 bounced back** — impossible in plum pudding! |
| The atom is mostly **filled** with positive matter | The atom is mostly **empty space** |
| No obvious feature at the centre | A dense, positively charged **nucleus** at the centre |

## 💡 Myth vs Fact

| Myth | Fact |
|------|------|
| Thomson **invented** the cathode-ray tube | Crookes invented it. Thomson used it as a **precision measurement instrument** — that was his genius |
| Thomson called it the "**plum pudding**" model | The name "plum pudding" was **unofficial** — Thomson described it as a "sphere of positive electrification" containing electrons. The pudding analogy came from British newspapers! |
| Thomson **discovered** the electron alone | His research students (especially **J.S.E. Townsend** and **C.T.R. Wilson**) did much of the experimental work — Thomson directed and theorised |
| The electron was **immediately accepted** | Some physicists were sceptical (e.g., Heinrich Hertz believed cathode rays were waves, not particles). The controversy lasted several years |
| Thomson's model was **quickly rejected** | The plum pudding model stood from **1904 to 1911** (7 years) — not bad for a pre-nuclear theory! |
| Thomson won the Nobel Prize **for the electron** | The 1906 Nobel was "for his theoretical and experimental investigations on the **conduction of electricity by gases**" — he didn't explicitly mention the electron in his Nobel lecture! |
| The word "**electron**" was coined by Thomson | **George Stoney** coined "electron" in **1891** to name the unit of charge found in electrolysis. Thomson called them "corpuscles" initially |
| The electron is an **indivisible point particle** | In the Standard Model, the electron is **fundamental** (no internal structure) — Thomson was right that it's indivisible, but not for the reasons he thought |

### Key Calculations

**1. Thomson's e/m value:**
```
e/m = 1.76 × 10¹¹ C/kg
If e = 1.60 × 10⁻¹⁹ C (Millikan, 1909):
m = e / (e/m) = 1.60 × 10⁻¹⁹ / 1.76 × 10¹¹ = 9.09 × 10⁻³¹ kg
```

**2. Velocity of the electron in Thomson's apparatus:**
Say E = 30,000 V/m, B = 0.01 T (balanced beam):
```
v = E/B = 30,000 / 0.01 = 3 × 10⁶ m/s (≈ 1% of the speed of light)
```

**3. Magnetic-field-only deflection:**
If B = 0.01 T and r = 0.05 m:
```
(e/m) = v/(Br) = 3 × 10⁶ / (0.01 × 0.05) = 6 × 10⁹ C/kg → (then adjust for geometry factor)

Thomson's actual calculation was more complex — he used integrated path length and averaged.
```

### Questions for Students

1. **Why** was Thomson's discovery of the electron revolutionary?
   - *Answer: It proved atoms are **not indivisible** — the first subatomic particle was found*

2. **Why** did Thomson conclude that cathode rays are the **same** for any metal?
   - *Answer: He measured the same e/m for cathodes made of aluminium, copper, iron, platinum — the particles are universal, not material-specific*

3. **Describe** how Thomson measured e/m.
   - *Answer: Balanced electric and magnetic fields → found v = E/B, then turned off E → measured circular deflection radius r → e/m = E/(B²r)*

4. **Why** was the plum pudding model replaced?
   - *Answer: Rutherford (1911) showed that alpha particles can bounce back from a gold foil — impossible if the positive charge were spread out → must be concentrated in a nucleus*
