📄 PDF Export (Python)

ทำอะไร: เปิดหน้าเว็บใน Headless Browser → จับภาพเป็น PDF

  • ใช้ Playwright เปิดเว็บแบบไม่มีหน้าจอ
  • อนิเมชั่น → ภาพนิ่ง frame สุดท้าย
  • ได้ PDF ที่ใช้ HTML/CSS เดียวกัน กับเว็บ
# Python — ตัวอย่าง: MD → PDF ด้วย Playwright from playwright.sync_api import sync_playwright def md_to_pdf(lecture_md: str, output: str): html = render_to_html(lecture_md) with sync_playwright() as p: browser = p.chromium.launch() page = browser.new_page() page.set_content(html) page.pdf(path=output, format="A4") browser.close()
