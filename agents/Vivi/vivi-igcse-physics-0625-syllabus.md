## 1️⃣ Motion, Forces and Energy

| หัวข้อหลัก | หัวข้อย่อย | รายละเอียด |
|---|---|---|
| 1.1 Physical quantities & measurement techniques | — | Units, rulers, vernier calipers, micrometer, scalars vs vectors |
| 1.2 Motion | — | Speed, velocity, acceleration, graphs (distance-time, velocity-time), equations of motion |
| 1.3 Mass and weight | — | Mass vs weight, gravitational field strength (g) |
| 1.4 Density | — | ρ = m/V, experiments for solids/liquids/irregular objects, floating & sinking |
| 1.5 Forces | 1.5.1 Effects of forces | Newton's Laws, friction, drag, Hooke's law (F = kx) |
| | 1.5.2 Turning effect | Moment = F × d, principle of moments |
| | 1.5.3 Centre of gravity | Stability, equilibrium |
| 1.6 Momentum | — | p = mv, conservation of momentum, collisions (elastic/inelastic) |
| 1.7 Energy, work & power | 1.7.1 Energy | KE = ½mv², GPE = mgh, conservation of energy |
| | 1.7.2 Work | W = Fd |
| | 1.7.3 Energy resources | Renewable (solar, wind, hydro, geothermal) vs non-renewable (fossil fuels, nuclear) |
| | 1.7.4 Power | P = W/t, P = Fv |
| 1.8 Pressure | — | p = F/A, hydraulics, atmospheric pressure, manometers, barometers |

## 2️⃣ Thermal Physics

| หัวข้อหลัก | หัวข้อย่อย | รายละเอียด |
|---|---|---|
| 2.1 Kinetic particle model | 2.1.1 States of matter | Solid, liquid, gas — arrangement, motion, forces |
| | 2.1.2 Particle model | Brownian motion, diffusion |
| | 2.1.3 Gases & absolute scale | Kelvin ↔ Celsius, pressure–temperature relationship |
| 2.2 Thermal properties & temperature | 2.2.1 Thermal expansion | Solids, liquids, gases — applications (bimetallic strip, thermometers) |
| | 2.2.2 Specific heat capacity | c = ΔQ/(mΔT), calorimetry experiments |
| | 2.2.3 Melting, boiling, evaporation | Latent heat, cooling effect of evaporation |
| 2.3 Transfer of thermal energy | 2.3.1 Conduction | Mechanism (free electrons, lattice vibration), good/poor conductors |
| | 2.3.2 Convection | Convection currents, applications |
| | 2.3.3 Radiation | Infrared, emission/absorption, surface effects |
| | 2.3.4 Consequences | Real-world applications (vacuum flask, cooling fins, etc.) |

## 4️⃣ Electricity and Magnetism

| หัวข้อหลัก | หัวข้อย่อย | รายละเอียด |
|---|---|---|
| 4.1 Simple phenomena of magnetism | — | Magnetic materials, poles, field lines, induced magnetism, magnetization |
| 4.2 Electrical quantities | 4.2.1 Electric charge | Positive/negative, attraction/repulsion, charging by friction |
| | 4.2.2 Electric current | I = Q/t, conventional current, ammeters |
| | 4.2.3 EMF & potential difference | Voltmeters, energy per unit charge |
| | 4.2.4 Resistance | Ohm's law, V = IR, resistors, I–V characteristics (ohmic vs non-ohmic) |
| | 4.2.5 Electrical energy & power | P = VI, E = Pt, kilowatt-hours |
| 4.3 Electric circuits | 4.3.1 Circuit diagrams & components | Symbols, cells, switches, bulbs, resistors |
| | 4.3.2 Series & parallel circuits | Current, voltage, resistance rules |
| | 4.3.3 Action & use of components | LDR, thermistor, diode, LED, relay |
| 4.4 Electrical safety | — | Fuses, earthing, circuit breakers, double insulation, dangers |
| 4.5 Electromagnetic effects | 4.5.1 Electromagnetic induction | Faraday's law, Lenz's law, factors affecting induced EMF |
| | 4.5.2 The a.c. generator | Slip rings, brushes, alternating current |
| | 4.5.3 Magnetic effect of a current | Right-hand grip rule, electromagnets |
| | 4.5.4 Force on a current-carrying conductor | Motor effect, Fleming's left-hand rule |
| | 4.5.5 The d.c. motor | Split ring commutator, brushes, practical uses |
| | 4.5.6 The transformer | Step-up/step-down, Vp/Vs = Np/Ns, power equation, energy losses |
